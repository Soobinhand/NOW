# NOW (Nickname On Web)
## NOW 란?
- 닉네임으로 활동하는 게시판입니다. 
- 닉네임은 익명이지만 중복 닉네임은 사용할 수 없으니 익명이 아닐지도...?
- 본인이 직접 게시판을 만들 수 있습니다.
- 게시글은 얼마든 작성해도 좋습니다.
- 홈 화면에는 본인이 찜을 한 게시판들이 있습니다.
- 궁금한 점 또는 그냥 하고 싶은 이야기들을 마음껏 써보세요!
